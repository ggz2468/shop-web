# 自製電子商務網站專案

### 成果網站
<a href="https://chun-hung.idv.tw" target="_blank">自製電子商務網站</a>

### 開發環境
1. Node.js 25.6.0
2. npm 11.8.0
3. Vue.js 3.5.32
4. Vite 8.0.4
5. Vue Router 5.0.4
6. Pinia 3.0.4
7. Axios 1.15.0
8. Bootstrap 5.3.8

### 安裝步驟
1. 下載專案與進入目錄
```bash
git clone git@github.com:ggz2468/shop-web.git
cd shop-web
```
2. 安裝必要套件
```bash
npm install
```
3. 設定環境變數
```bash
cp .env.example .env
```
4. 啟動開發伺服器
```bash
npm run dev
```
5. 本地開發網址: http://localhost:5173
